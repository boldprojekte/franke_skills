# Skills

This index is maintained manually for now. When you add, remove, or rename a skill under `skills/`, update this file in the same change: add a row to the table of its category folder (creating a new category section if needed), link the skill name to its `SKILL.md`, and copy the one-line `description` from that file's YAML frontmatter, truncated to 140 characters at a word boundary if longer.
