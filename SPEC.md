Add a validation script scripts/check_skills.py to this repository (stdlib-only Python). It walks skills/ recursively, and for every SKILL.md verifies the YAML frontmatter contains a non-empty name and a non-empty description field. It prints one line per violation (path plus what is missing) and exits with code 1 if any violation exists, 0 otherwise, printing OK plus the number of checked skills. Do not modify anything else.
